About PyFiddle

PyFiddle is a free lightweight Python IDE to run and share Python scripts with some nifty features, created with love by pythonista Priyank Pulumati Website Policy

{% csrf_token %}
  • Shortcuts:
  • Clear Editor : Control/Command + Backspace
  • Run Fiddle : Control/Command + Enter
  • Save Fiddle : Control/Command + S
  • Text Editor: Default common shortcuts (Select all, copy, paste, undo)
  • Supports:
  • Saving and Sharing Fiddles for free
  • Pip installation of Pakcages
  • Arguments to scripts
  • File uploads
  • Limitations:
  • Compute Time: Max of ~ 4 minutes, 256 MB of RAM
  • Pip packages limited to 5 and run without need for compiling
  • Python 2.7 (as of now)
  • File size upto 1MB per file, upto 10 files
